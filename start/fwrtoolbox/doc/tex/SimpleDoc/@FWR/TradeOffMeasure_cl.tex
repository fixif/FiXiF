\begin{command}[@FWR/TradeOffMeasurecl]{TradeOffMeasure\_cl}	\desc{Purpose}Compute a (pseudo) tradeoff closed-loop measurewith the MsensH\_cl, MsensPole\_cl, RNG\_cl	\desc{Syntax}\matlab{M = TradeOffMeasure\_cl( R, Sysp, MH, MP, MRNG)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{M} &  measure value                                                           \\\matlab{R} &  FWR object                                                              \\\matlab{Sysp} &  plant system (\matlab{ss} object)                                           \\\matlab{MH,MP,MRNG} &  optimal value of the MsensH\_cl, MsensPole\_cl and RNG\_cl measure\\		\end{tabular}	\desc{Description}Even if a tradeoff measure like this one is not the best solution to multi-objective optimal realization, it is interesting to look for a realization that is \I{good enough} for the three measures $\bar{M}_{L_2}^W$, $\bar{\Psi}$ and $\bar{G}$.This (pseudo) tradeoff measure is defined by\begin{equation}\bar{TO}(Z) \triangleq \frac{\bar{M}_{L_2}^W(Z)}{\bar{M}_{L_2}^{W\ opt}} + \frac{\bar{\Psi}(Z)}{\bar{\Psi}^{opt}} + \frac{\bar{G}(Z)}{\bar{G}^{opt}}\end{equation}where $\bar{M}_{L_2}^{W\ opt}$, $\bar{\Psi}^{opt}$ and $\bar{G}^{opt}$ are the optimal values for these respective measures.\desc{See also}\funcName[@FWR/MsensHcl]{MsensH\_cl}, \funcName[@FWR/MsensPolecl]{MsensPole\_cl}, \funcName[@FWR/RNGcl]{RNG\_cl}	\desc{References}\cite{Hila08b} T. Hilaire, P. Chevrel, and J. Whidborne. Finitewordlength controller realizations using the specialized implicit form. Technical Report RR-6759, INRIA, 2008.\end{command}
