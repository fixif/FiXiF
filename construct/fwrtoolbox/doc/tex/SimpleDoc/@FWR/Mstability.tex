\begin{command}[@FWR/Mstability]{Mstability}	\desc{Purpose}Compute the closed-loop pole sensitivity stability related measurefor a FWR object	\desc{Syntax}\matlab{M = Mstability( R, Sysp, moduli)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{M} &  pole sensitivity measure                                                                                        \\\matlab{R} &  FWR object                                                                                                      \\\matlab{Sysp} &  plant system (ss object)                                                                                     \\\matlab{moduli } &  1 (default value) : compute $\dd{\abs{\bar\lambda}}{Z}$ (the sensitivity of the moduli of the eigenvalues)\\\matlab{} &  0 : compute $\dd{\bar\lambda}{Z}$ (without the moduli)                                                           \\		\end{tabular}	\desc{Description}The Pole Sensitivity Stability related Measure (PSSM) is defined by\begin{equation}\mu_1(Z) \triangleq \underset{1 \leq k \leq \np+n}{\min} \frac{ 1- \abs{\strut \bar\lambda_k} }{ \norm{W_Z}_F \norm{ \dd{\abs{\bar\lambda_k}}{Z} \times W_Z }_F }.\end{equation}This measure evaluates how a perturbation, $\Delta$, of the parameters, $Z$, can cause instability. It is determined by how close  the eigenvalues of $\bar{A}$ are to the unit circle and by how sensitive they are to the controller parameter perturbation.\\See \funcName[@FWR/MsensPolecl]{MsensPole\_cl} for thecomputation of $\dd{\abs{\bar\lambda_k}}{Z}$.\desc{See also}\funcName[@FWR/MsensPolecl]{MsensPole\_cl}, \funcName[@FWS/Mstability]{Mstability}	\desc{References}\cite{Hila08b} T. Hilaire, P. Chevrel, and J. Whidborne. Finite wordlengthcontroller realizations using the specialized implicit form.Technical Report RR-6759, INRIA, 2008.\end{command}
