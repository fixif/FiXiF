\begin{command}[@FWR/ss]{ss}	\desc{Purpose}Convert a FWR object into a ss (state-space) object (equivalent state-space)	\desc{Syntax}\matlab{S = ss(R,Te)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{S} &  ss object          \\\matlab{R} &  FWR object         \\\matlab{Te} &  period (default=1)\\		\end{tabular}	\desc{Description}Give the equivalent state-space (\matlab{ss} object). It is defined with matrices$A_Z$, $B_Z$, $C_Z$ and $D_Z$ (see eq. \eqref{eq:defAZandBZ} and \eqref{eq:defCZandDZ}).	\desc{Example}\matlab{>>bode(ss(R));}\\ where \matlab{R} is a FWR object, allows to plot its Bode frequency response.\desc{See also}\funcName[@FWR/tf]{tf}, \funcName[@FWS/ss]{ss}\end{command}
