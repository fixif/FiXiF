\begin{command}[@FWS/set]{set}	\desc{Purpose}Set some properties of a FWS object	\desc{Syntax}\matlab{S = set(S, propName, value)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{S} &  FWR object                      \\\matlab{propName } &  name of the property    \\\matlab{value } &  new value for this property\\		\end{tabular}	\desc{Description}This function is most of the time called by \funcName[@FWS/subsasgn]{subsasgn}.\\The initial realization \matlab{Rini} can be changed (if the size doesn't change), so can the parameters of the structuration. It is not allowed to modified other fields.\desc{See also}\funcName[@FWS/get]{get}, \funcName[@FWS/subsasgn]{subsasgn}, \funcName[@FWR/set]{set}\end{command}
