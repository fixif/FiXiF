\begin{command}[@FWS/genCostFunctionS]{genCostFunctionS}	\desc{Purpose}Generic cost function for optimization of a FWS.Here the measure is computed direclty from the U,Y,W (andother values computed one time and stored in data) in order to decreasethe computational time(in \matlab{genCostFunctionR}, it's computed from the new realization R)	\desc{Syntax}\matlab{[cost\_value, cost\_flag] = genCostFunctionS( S, x, freeparams, l2scaling, Umax, delta, measureFun, ...)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{cost\_value} &  value of the measure for parameter x                           \\\matlab{cost\_flag} &  0 if parameter x is incorrect                                   \\\matlab{} &  1 otherwise                                                              \\\matlab{S} &  FWS object                                                              \\\matlab{x} &  vector of parameter used by optimiser (ASA, fminsearch, ...)            \\\matlab{freeparams} &  vector that indicates which parameters are free to be optimized\\\matlab{l2scaling} &  boolean - tell if R is $L_2$-scaled                             \\\matlab{Umax} &  magnitude value for the input - used for the $L_2$-scaling           \\\matlab{delta} &  delta parameter for $L_2$-scaling                                   \\\matlab{measureFun } &  handle of the measure function                                \\\matlab{...} &  the extra parameters are given to the measure function                \\		\end{tabular}	\desc{Description}\begin{center}\I{Internal function}\end{center}This function is a generic cost function. From the vector $x$, itrebuilds the matrices $\mt{U}$, $\mt{Y}$ and $\mt{W}$ (thanks to the \matlab{UYWfun} function) and thencompute the associated cost value.\\In \funcName[@FWS/genCostFunctionS]{genCostFunctionS}, a new realization is directly computed from the parameters' value(thanks to the \matlab{Rfun} function), and used to compute the cost value.\desc{See also}\funcName[@FWS/optim]{optim}, \funcName[@FWS/genCostFunction]{genCostFunction}, \funcName[@FWS/genCostFunctionR]{genCostFunctionR}\end{command}
