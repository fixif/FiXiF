\begin{command}[@FWS/RNG]{RNG}	\desc{Purpose}Compute the open-loop Roundoff Noise Gain for a FWS.The computation is based on the UYW-transform	\desc{Syntax}\matlab{G = RNG(S,U,Y,W)}	\desc{Parameters}		\begin{tabular}{l@{\ :\ }p{9cm}}\matlab{G} &  roundoff noise gain         \\\matlab{S} &  FWS object                  \\\matlab{U,Y,W } &  transformation matrices\\		\end{tabular}	\desc{Description}This function computes the open-loop Roundoff Noise Gain for a FWS. The computation is based on the $\mt{UYW}$-transform.\\It is based on the \funcName[@FWR/RNG]{RNG} function.\\The matrix $d_Z$ is computed once and stored in the \matlab{dataMeasure} field.\desc{See also}\funcName[@FWR/RNG]{RNG}	\desc{References}\cite{Hila07c} T. Hilaire, D. Ménard, and O. Sentieys. Roundoff noise analysis of finite wordlength realizations with the implicit state-space framework. In 15th European Signal Processing Conference (EUSIPOC'07), September 2007.\end{command}
